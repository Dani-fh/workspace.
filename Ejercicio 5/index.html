SISTEMAS DE MEDIDA EN CSS

En CSS, los sistemas de medida permiten definir tamaños, espacios y proporciones de los elementos HTML. Saber cuándo usar unidades absolutas o relativas es clave para lograr un diseño adaptativo y coherente.

Unidades Absolutas

Las unidades absolutas tienen un valor fijo sin importar el dispositivo o pantalla. Ejemplos: px, cm, in, pt, y pc.

Unidades Relativas

Las unidades relativas dependen del entorno del elemento, como el tamaño de fuente o el tamaño del viewport. Ejemplos: %, em, rem, vw, vh.
